## 파워쉘로 깃허브 폴더 클론 복제

- 원하는 폴더로 이동 후 해당 명령어 파워쉘에서 실행
```
git clone [원하는 프로젝트가 있는 내 깃허브 주소 경로]
```


## 윈도우 파워쉘 최신버전인지 확인 후 설치

```
winget search --id Microsoft.PowerShell

winget install --id Microsoft.PowerShell --source winget

winget install --id Microsoft.PowerShell.Preview --source winget
```

## 깃허브 코드 올리는 법

- git add . -> 모든 파일을 올리겠다.
- git add [파일명] -> 특정 파일만 올리겠다.

- add -> commit -> push 는 하나의 작동
- push: 깃허브 사이트에 올리는 역할
