## 개요

GitHub에서 파일을 폴더별로 정리하고 싶을 때 사용하는 방법이다. GitHub는 빈 폴더를 허용하지 않기 때문에 폴더 생성 시 파일이 최소 1개 필요하다.

## 실습 과정

**1. Repository에서 원하는 폴더 선택**

**2. Add file → Create new file 선택**

**3. 파일명 칸에 `폴더명/1` 형식으로 입력**

```
ex) Cpp/1
```

**4. 우측 상단 Commit changes 버튼 클릭 → 한 번 더 확인 후 완료**

**5. 생성된 파일(1) 확인 후 파일명 변경해서 사용**

**6. 기존 파일 이동 시 파일 내 폴더명만 변경**

```
ex) Cpp → 다른 폴더명
```

## 요약

- GitHub 빈 폴더 불가 → 폴더 생성 시 파일 1개 필수
- 파일 이동은 파일명의 폴더 경로만 수정하면 됨
