## 2. Metasploitable 2 네트워크 설정

```
sudo vi /etc/network/interfaces
```

```
# 기본 루프백 설정 (건드리지 마세요)
auto lo
iface lo inet loopback

# eth0 설정 수정
auto eth0
iface eth0 inet static        # dhcp를 static으로 변경
    address 192.168.100.20    # 원하는 IP
    netmask 255.255.255.0     # 서브넷 마스크
```

## 3. RHEL 9.7

```
# 인터페이스 이름 확인
nmcli device

# IP 주소 및 서브넷 설정
nmcli connection modify ens160 ipv4.addresses 192.168.100.40/24 ipv4.method manual

# 설정 적용 및 활성화
nmcli connection up ens160

# 확인
ip addr show ens160
```
