# 네트워크 보안 [과목2]

### DDoS 공격 3유형 & 방화벽 종류

	1. Volumetric
	2. Protocol
	3. Application Layer
	4. 패킷 필터링
	5. 상태 추적
	6. 애플리케이션 게이트웨이

### IDS[탐지] vs IPS[차단]

	 IPS가 더 좋음
	 IDS는 세션을 유지, 반면 IPS는 세션을 끊어줌
		-> 1라인 배치 ---ㅁ---
	IDS에는 오용탐지[기존패턴], 이상탐지[다른패턴] 있음

### IPSec

	 AH, IKE, ESP

### SSL/TLS 동작 & 무선 보안 프로토콜

	 WPA2 -> WPA3 최신 사용, 둘 다 같이 사용
	 PKI 존재, 인증서 검증

# 애플리케이션 보안[과목3]

### SQL 인젝션

	SQL 코드를 삽입해 DB를 위,변조
		-> 'OR '1' ='1 이걸로 TRUE 값을 만들어서 인증 우회
	XSS - 저장형: 악성 스크립트를 게시판, 댓글에 저장
	-> 다른 사용자 방문 시 자동 실행
	XSS - 반사형: URL 파라미터에 스크립트 삽입
	-> 클릭하면 해당 브라우저에서 실행

### CSRF, 인증 취약점, 기타 웹

	CSRF: 로그인된 사용자가 정상 서버에 위조 요청 보냄, 피해자는 모름
	세션 하이재킹
	파일 업로드
	디렉터리 트래버설 등 존재
