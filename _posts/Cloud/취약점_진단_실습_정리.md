### WinRM 활성화 (진단 전 필수)

```powershell
# 네트워크 프로파일 Private으로 변경
Set-NetConnectionProfile -InterfaceAlias "Ethernet0" -NetworkCategory Private

# WinRM 활성화
Enable-PSRemoting -Force
winrm set winrm/config/client '@{TrustedHosts="*"}'
winrm set winrm/config/service/auth '@{Basic="true"}'
winrm set winrm/config/service '@{AllowUnencrypted="true"}'

# Administrator 계정 활성화
Enable-LocalUser -Name "Administrator"
Set-LocalUser -Name "Administrator" -Password (ConvertTo-SecureString "Tj0tlf.0505" -AsPlainText -Force)

# 방화벽 포트 허용
netsh advfirewall firewall add rule name="WinRM-HTTP" dir=in action=allow protocol=TCP localport=5985

# W32Time 서비스 시작 (NTP 점검용)
Start-Service W32Time
Set-Service W32Time -StartupType Automatic
```

### WinRM 원복 (진단 후 필수)

```powershell
Disable-PSRemoting -Force
winrm set winrm/config/service/auth '@{Basic="false"}'
winrm set winrm/config/service '@{AllowUnencrypted="false"}'
Disable-LocalUser -Name "Administrator"
```

## 6. Solaris 11.4 네트워크 설정

```bash
# 기존 주소 삭제
ipadm delete-addr net0/v4

# 고정 IP 설정
ipadm create-addr -T static -a 192.168.100.60/24 net0/v4
```

### nova 계정 설정 (진단 전 필수)

```bash
# nova 계정 생성
useradd -m -d /export/home/nova -s /bin/bash nova
passwd nova

# sudo 권한 부여 (visudo 실행 후 아래 줄 추가)
visudo
# nova    ALL=(ALL) ALL
```
