# 가상 네트워크 설계

	VMnet11이라는 가상 스위치 생성
	- 네트워크 방식: nat (host 통신 가능, 가상머신끼리 가능, 외부 인터넷 가능)
	- 서브넷 대역: 192.168.20.0 / 24
	- 게이트웨이: 192.168.20.2 (nat)


![](Pasted%20image%2020260327155311.png)
# Kali 설정

	ip 주소: 192.168.20.25
	gateway: 192.168.20.2(nat)

# Metasploitable2 설정

![](Pasted%20image%2020260327155327.png)

	ip 주소: 192.168.20.26
	gateway: 192.168.20.2
	
	nano 편집기를 사용해 수정완료
	`sudo nano /etc/network/interfaces`
	저장: ctrl + o
	나가기: ctrl + x
	
	수정내용: dhcp -> static
	ip주소: 192.168.20.26
	넷마스크: 255.255.255.0
	게이트웨이: 192.168.20.2
	dns: 8.8.8.8
	
	수정 후 명령어: networking restart
