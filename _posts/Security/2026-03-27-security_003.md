# OWASP Top 10

	웹 앱에 대한 취약점 Top 10
	
	용어: 
	CVE[Common Vulnerabitities and Exposures]: 공통 취약점 및 노출, 예시: CVE-2021-44228
	CWE[Common Weakness Enumeration]: 공통 약점 목록 
	CVSS[Common Vulnerability Scoring System]: 공통 취약점 평가 시스템, 0~10까지 위험도 세분화
	CCE[Common Configuration Enumeration]: 시스템의 취약한 설정에 대한 점검
	 

### 1. Broken Access Control

	CWE: 40
	
	IDOR, 강제브라우징, 수직권한 상승, JWT 변조

### 2. Security Misconfiguration

	CWE: 16
	
	기본 자격증명 악용, 디버그 정보 노출, 불필요한 서비스 노출, 보안 헤더 미설정

### 3. Software Supply Chain Failures

	CWE: 5
	
	악성 패키지 삽입, 의존성 혼란 공격, 빌드 파이프라인 침해, 타이포스쿼팅

### 4. Cryptographic Faulures

	CWE: 32
	
	평문 전송 가로채기, 취약한 해시 크래킹, 하드코딩된 시크릿, 취약한 암호화 모드
