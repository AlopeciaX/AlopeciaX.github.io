1.FTP(File Transfer Protocol)
	1.1. 대용량 파일 전송 서비스
	1.2. Protocol: TCP
	1.3. Port: 21(제어 및 인증)
	1.4. Data 전송 Port:
		1.4.1. Active(능동): 클라이언트가 포트를 결정, 서버쪽은 20
			1.4.1.1. 클라이언트가 데이터 전송 포트를 결정해서 서버에 통보
			클라이언트가 알려준 포트로 서버가 데이터를 전송
			화이트보드 현상 발생: Client측 포트를 알지 못해서 방화벽에서 차단
		1.4.2. Passive(수동): 데이터 전송 포트 서버가 결정(관리자가 구성)
			1.4.2.1. 클라이언트가 서버쪽에 데이터 전송을 요청하면 서버는 데이터 전송포트를 Client에게 통보
			클라이언트가 서버의 해당 포트에 접속해서 데이터를 전송해 감.
	1.5. 구성
		1.5.1. # dnf install -y vsftpd
	1.6 설정 파일 위치
		1.6.1 /etc/vsftpd/vsftpd.conf

## FTP 실습

1. 사용자 계정 x, y / w10-1 - x, w11-1 - y
2. ftp는 active mode
3. 배너는 파일로, 경로는 /ftp/ban
4. chroot는 x에만 적용, list로 작성, 파일은 /ftp/chroot 저장
5. anonymous계정 사용안함
6. xferlog는 /ftp/xferlog로 작성
7. 3분동안 가만히있으면 로그아웃
8. data 전송중이라도 30초 넘으면 로그아웃
9. 나머지 기본값
