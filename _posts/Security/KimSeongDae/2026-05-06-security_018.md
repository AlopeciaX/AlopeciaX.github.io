	2. Protocol: UDP
		1. TCP 사용:
			1. 전송하는 DATA가 512Byte 이상(전세계 root dns server가 13대로 제한)
			2. 영역전송(Zone Transfer)
	3. port: 53
	4. 설정파일
		1. /etc/named.conf
		2. /etc/named.rfc192.zones
		3. /var/named 디렉토리에 1.4.2. 지정한 영역파일 생성(레코드값 생성)
