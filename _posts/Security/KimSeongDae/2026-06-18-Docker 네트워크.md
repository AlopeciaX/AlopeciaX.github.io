##### docker network 통신확인

![](../../../assets/images/Security/KimSeongDae/2026-06-18-Docker%20네트워크/file-20260618093411319.png)

브릿지는 개수 증가 가능, host는 1개만

![](../../../assets/images/Security/KimSeongDae/2026-06-18-Docker%20네트워크/file-20260618093520766.png)

동일 네트워크기 떄문에 서로 통신이 됨

```bash
docker network ls
docker exec a1 ping 172.17.0.3
docker exec a1 ping 172.17.0.2

docker rm -f $(docker ps -aq) #지우는 명령어
```
