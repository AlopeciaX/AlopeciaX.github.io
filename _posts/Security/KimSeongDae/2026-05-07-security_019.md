**1. 주영역 만들기 (rocky9-1)**
```bash
vi /etc/named.conf #any any 로 수정

vi /etc/named.rfc1912.zones

cp /var/named/{named.localhost,1} #named.localhost: 정방향
cp /var/named/{named.loopback,2} #named.loopback: 역방향
```
- URL → IP 주소: 정방향
- IP 주소 → host: 역방향 (nslookup 했을 때 unknown으로 나옴)
	역방향이 문제 있더라도 정방향이 정상이면 문제 없음

![](../../../assets/images/Security/KimSeongDae/2026-05-06-security_019/file-20260507095536679.png)
`vi /etc/named.rfc1912.zones`

- allow-update: master로부터 데이터를 가져가도록 허용하는 IP 주소들
- "0.0.10.in-addr.arpa": 맨 마지막 값은 입력할 수 없음 → 레코드 값에 들어가야 되므로

```bash
vi /var/named/1
```

![](../../../assets/images/Security/KimSeongDae/2026-05-06-security_019/file-20260507100251735.png)
`vi /var/named/1`

- A: Ipv4꺼라 작성
- AAAA: Ipv6꺼라 필요없으니 제거
	위의 설정한 내용은 아래에도 똑같이 생성

```bash
chmod o+r /var/named/{1,2} #읽기 모드 꼭 추가해주기

systemctl enable --now named
systemctl restart named
```
