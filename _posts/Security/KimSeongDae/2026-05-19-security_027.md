**NMAP**

```bash
nmap -sT -v -O -p0-65535 10.0.0.31 > metas3.txt
```

	0-65535까지 tcp인 모든 포트 탐색
	리디렉션 기호로 탐색 결과를 꼭 저장해주는게 좋음
	

```bash
snmp-check 10.0.0.31
```
