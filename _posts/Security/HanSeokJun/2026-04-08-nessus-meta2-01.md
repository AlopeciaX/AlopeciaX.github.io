# nmap

- Kali 명령어

	
	nmap [ip]
	nmap -sV -sC [ip]
	nmap -sV [ip] > scan.res
	
	nmap으로 어떤 포트가 취약점이 있는지 확인 한 다음
	
	searchsploit vsftpd 2.3.4
	https://www.exploit-db.com/ 해당 사이트를 참조해서 취약점을 찾아준다.
	![](../../../assets/images/Security/HanSeokJun/2026-04-08-nessus-meta2-01/file-20260408113345001.png)
	![](Pasted%20image%2020260408113533.png)
	
	searchsploit으로 취약점 목록을 추출해준다.
	
	find / -name 49797.py 2>/dev/null
	![](../../../assets/images/Security/HanSeokJun/2026-04-08-nessus-meta2-01/file-20260408113950008.png)
	해당된 파일의 절대경로를 보여준다.
	
	
	해당 취약점에 대한 보안공격을 검색
	ex) vsftpd 2.3.4 에 대한 백도어 공격 -> 실행가능
	
	cp /usr/share/exploitdb/exploits/unix/remote/49757.py .exploit.py
	![](../../../assets/images/Security/HanSeokJun/2026-04-08-nessus-meta2-01/file-20260408115323009.png)
	원하는 폴더로 가서 exploit.py를 복사해준다.
	
	![](../../../assets/images/Security/HanSeokJun/2026-04-08-nessus-meta2-01/file-20260408122816011.png)
	오류가 났는데 6200포트에 뭐가 문제가 있어보인다. nmap으로 해당 포트를 확인
	nmap -p 6200 [대상ip]
	python .exploit.py [대상ip]
	![](../../../assets/images/Security/HanSeokJun/2026-04-08-nessus-meta2-01/file-20260408122537010.png)
	상대방의 취약점을 이용해 접속한 상태이다.
	
	![](../../../assets/images/Security/HanSeokJun/2026-04-08-nessus-meta2-01/file-20260408152746012.png)
	
	매우 취약한 취약점 -> 그냥 접속 가능
