리눅스는 하나만 공부해놓으면 나머지는 쉬움

1. 가상화
	- Hardware 자원을 추상화(CPU, Memory, HDD, NIC)해서 Virtual machine에 제공하는 기술
	- 종류
		- 서버가상화
		- 데스크탑 가상화(다양한 os를 하나로 통합해서 사용 가능(ex) 윈도우, 리눅스, mac ...-> 하나로)
		- App 가상화
	- 데스크탑 가상화
		- Citrix Xendesktop
		- VMware
	- App 가상화
2. 형태
	- Type1, Hypervisor (hardware -> hypervisor -> os들)
		- VMware ESXi
		- Cirtrix Xenserver
		- Hyper-v
	- Type2, Hosted (hardware -> os -> hypervisor -> os들)
		- VMware workstation
		- Virtualbox

기존 os -> hypervisor -> container(docker)(쿠버네티스) -- 이거 쿠버네티스 할 줄 알아야함
