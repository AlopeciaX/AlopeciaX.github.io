시프트 연산자
임의로 만든 함수는 main함수위에 데이터 타입을 선언해줘야 한다.


1. 함수 헤더 선언 (example.h 형식)

```cpp
#include <stdio.h>

// 권한 비트 정의 (각각 1비트 차지)
#define READ    (1 << 0)  // 0001
#define WRITE   (1 << 1)  // 0010
#define EXEC    (1 << 2)  // 0100
#define ADMIN   (1 << 3)  // 1000

// 함수 원형 선언
void printPermissions(int perm);
int hasPermission(int perm, int check);
int addPermission(int perm, int add);
int removePermission(int perm, int remove);
int togglePermission(int perm, int toggle);

// --- 함수 구현부 ---

// 현재 권한 상태 출력
void printPermissions(int perm) {
printf("현재 권한: ");
if (perm & READ)  printf("READ ");
if (perm & WRITE) printf("WRITE ");
if (perm & EXEC)  printf("EXEC ");
if (perm & ADMIN) printf("ADMIN ");
printf("\n");
}

// 특정 권한 체크
int hasPermission(int perm, int check) {
return (perm & check) != 0;
}

// 권한 추가
int addPermission(int perm, int add) {
return perm | add;
}

// 권한 제거
int removePermission(int perm, int remove) {
return perm & (~remove);
}

// 권한 토글
int togglePermission(int perm, int toggle) {
return perm ^ toggle;
}
```

**C 언어**는 1972년 벨 연구소의 데니스 리치에 의해 개발된 **범용 시스템 프로그래밍 언어**입니다. 현대의 수많은 언어(C++, Java, Python 등)의 뿌리가 된 '프로그래밍의 근본'이라 불립니다.

---

## 1. 주요 특징

- **저수준 제어:** 하드웨어에 직접 접근이 가능하며 메모리를 정밀하게 관리할 수 있습니다.
    
- **높은 효율성:** 컴파일 언어로 실행 속도가 매우 빠르고 리소스를 적게 소모합니다.
    
- **이식성:** CPU 아키텍처에 상관없이 C 컴파일러만 있다면 어디서든 코드를 빌드하고 실행할 수 있습니다.
    
- **절차지향:** 코드가 작성된 순서대로 실행되는 직관적인 구조를 가집니다.
    

## 2. 장점과 단점

|**구분**|**내용**|
|---|---|
|**장점**|시스템 성능 최적화 가능, 운영체제(OS) 및 임베디드 개발에 필수적.|
|**단점**|포인터 등 메모리 관리가 까다롭고, 실수 시 시스템 오류(Crash) 유발이 쉬움.|

## 3. 주요 활용 분야

1. **운영체제 제작:** Windows, Linux, macOS 등의 커널 핵심 코드가 C로 작성됨.
    
2. **임베디드 시스템:** 가전제품, 자동차 제어 장치 등 저사양 하드웨어 제어.
    
3. **컴파일러/인터프리터:** Python 같은 다른 프로그래밍 언어 자체를 만드는 데 사용.

## 4. 기본 코드 예시

C

```
#include <stdio.h> // 표준 입출력 라이브러리 포함

int main() {
    printf("Hello, World!\n"); // 화면에 출력
    return 0; // 프로그램 정상 종료
}
```

현재 진행 중인 클라우드 보안이나 서버 관리 학습에서도, 리눅스 커널이나 네트워크 장비의 동작 원리를 깊게 이해하려면 C 언어의 기본 구조를 아는 것이 큰 도움이 됩니다.
